# 进入服务器

本服务器需要专用客户端！

由于HMCL的未知原因会崩溃，暂时不放进去自动更新的客户端中！

# 下载客户端

在群文件-群服务器客户端文件夹

下载：夜阑星空群服务器模组客户端

下载完之后，解压，你会得到以下文件

# 运行客户端

打开Nsiso启动器

先进入设置，修改下载源为MCBBS（一会要用到）

然后选择版本，输入你的群服务器邮箱

然后，启动游戏，输入你的密码

这个时候，需要下载资源库文件，点击下载

然后等待游戏启动即可（如果启动器出错，就再试一次）

# 修改内存的方法

默认是4gb，你可以自己修改

进入设置

最小建议2G或者更多，最大最起码给3G吧

# 进入游戏

选择节点，进去

你会发现，就是平时生存服的主城。。。

（好吧，告诉你，主城版本就是1.12，使用了插件，1.16.5也进得去）

使用指南针选择模组服务器即可进去！
