# 领地传送指令

权限传送权限对所有人开放！

但领地内部权限，请询问领地主人要使用权！

或者在领地主人同意的情况下，找服务器管理员要权限

领地传送：

```
/res tp 领地
```

查询所有领地：

```
/res listall
```
